
# Streaming Intelligence Lakehouse

**Streaming Intelligence Lakehouse** is a production-style data engineering project that simulates the analytics platform of a video streaming company similar to Netflix.

The platform processes high-volume playback event data using **Databricks, PySpark, and Delta Lake** and transforms raw streaming activity into analytics-ready datasets that support:

- Viewer engagement analytics  
- Recommendation system features  
- Content performance monitoring  
- Platform usage insights  

The system follows a **Medallion Lakehouse Architecture (Bronze → Silver → Gold)** to demonstrate how modern streaming platforms build scalable data infrastructure.

# Architecture Overview

The platform processes viewer activity events generated by streaming devices such as TVs, mobile applications, and web browsers.

These events flow through a **Lakehouse Medallion Architecture**.

```mermaid
flowchart TD

A[Streaming Applications<br>TV | Mobile | Web] --> B[Playback Event Logs]
B --> C[Raw Data Lake]

C --> D[Bronze Layer<br>Raw Event Data]
D --> E[Silver Layer<br>Cleaned & Standardized Data]
E --> F[Gold Layer<br>Analytics & Feature Tables]

F --> G[BI Dashboards]
F --> H[Recommendation Systems]
```

---

# Layer Responsibilities

## Bronze Layer
- Raw playback events
- Minimal transformations
- Ingestion metadata

## Silver Layer
- Data cleaning and normalization
- Duplicate removal
- Sessionization
- Standardized schemas

## Gold Layer
- Aggregated analytics datasets
- Recommendation feature tables
- Content performance metrics

# Playback Events

Playback events represent individual interactions between a user and streaming content.

## Example Schema

| Column | Type | Description |
|---|---|---|
| event_id | string | Unique identifier for the playback event |
| user_id | string | Unique identifier for the viewer |
| session_id | string | Identifier grouping events within a viewing session |
| content_id | string | Identifier for the movie or show |
| event_type | string | Type of playback event |
| event_timestamp | timestamp | Timestamp when the event occurred |
| device_type | string | Device used for playback |
| platform | string | Application platform |
| country | string | Country of the viewer |
| watch_seconds | integer | Seconds watched |
| completion_pct | float | Percentage of content completed |
