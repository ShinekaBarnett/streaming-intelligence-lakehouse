# Streaming Intelligence Lakehouse
## A Databricks/PySpark data platform simulating Netflix-style streaming analytics using playback event pipelines, medallion architecture (Bronze/Silver/Gold), and Delta Lake to power engagement analytics and recommendation features. Streaming Intelligence Lakehouse is a production-style data engineering project that simulates the core analytics platform of a video streaming company similar to Netflix

## The platform processes high-volume playback event data using Databricks, PySpark, and Delta Lake and transforms raw streaming activity into analytics-ready datasets supporting

### viewer engagement analytics

### recommendation system features

### content performance monitoring

### platform usage insights

## The architecture follows a Lakehouse Medallion pattern (Bronze → Silver → Gold) to demonstrate how modern streaming companies build scalable data platforms

## Project Goals

### This project demonstrates how a streaming platform processes large volumes of event data to support business intelligence and machine learning workloads

#### Key goals

##### simulate real streaming platform data pipelines

##### demonstrate large-scale ETL with PySpark

##### implement lakehouse architecture using Delta Lake

##### build analytics and recommendation feature datasets

##### simulate production data engineering practices

## Architecture Overview
